Peer Definitions:

1. Core PB: Goldman Sachs Bank USA, UBS Bank USA.

2. All Peers: US G-SIB Credit Intermediaries + UBS.


Color Coding: Green = Favorable vs Peers | Red = Unfavorable vs Peers | Brown = Neutral (within 5bps)


Methodology:

Risk-Adj ACL Ratio: Total ACL / (Gross Loans - SBL). Removes low-risk SBL to show coverage on core credit.

CRE NPL Coverage: CRE-Specific ACL / CRE Nonaccrual Loans. ($ Reserved per $ of Bad Loans).

Peer Average Calculation: Core Private Bank Peers (Goldman Sachs + UBS) - Weighted by Gross Loans.

Segment Metrics: Weighted by respective segment balances (CRE metrics by CRE Balance, Resi metrics by Resi Balance).
