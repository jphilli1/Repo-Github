AI-NATIVE SPLIT-RAG SYSTEM (v2.0) - PROJECT FILE GUIDE

FILE MANIFEST & MAPPING

Use this guide to know which code block to paste into which file.

[Setup & Configuration]

init_project.py       -> [Project Initialization Script]
Run this FIRST to build folders.

install_deps.py       -> [Install via Python Script]
Run this to install dependencies if Terminal is locked.

requirements.txt      -> [Tier 1 Dependencies]
List of libraries (Docling, Pandas, etc.).

config.json           -> [Operational Configuration]
Main settings file.

rules.json            -> [Entity Extraction Rules]
Regex for Borrowers, Dates, Doc Types.

bootstrap.bat         -> [Windows Entry Point]
Startup script.

[Tier 1: The Factory]

schema_v2.py          -> [Data Contract]
Defines Pydantic models & IDs.

relationship_manager.py -> [Relationship Manager]
The Gatekeeper/Indexer.

extractor.py          -> [Tier 1 Extraction Engine]
The main PDF processor.

[Tier 2: The Consumer]

copilot_tier2.py      -> [Tier 2 Sandbox Script]
The chatbot script (StdLib + Pandas only).

[Testing & Validation]

tests/conftest.py           -> [Test Fixtures]
Shared setup for tests.

tests/test_determinism.py   -> [T-001 Validation]
Tests for deterministic output.

tests/test_disambiguation.py -> [T-002 Validation]
Tests for entity resolution.

tests/test_data_mart.py     -> [T-003 Validation]
Tests for Excel/CSV export.

tests/test_tier2_retrieval.py -> [Tier 2 Function Tests]
Tests for retrieval logic.

scripts/validate_deployment.py -> [Deployment Validator]
Checks for forbidden imports.

[Runners]

scripts/run_tests.bat       -> [Windows Test Runner]

scripts/run_tests.sh        -> [Linux/Mac Test Runner]

[Documentation]

docs/DEPLOYMENT_GUIDE.md    -> [Deployment Guide]

ORDER OF OPERATIONS

STEP 1: INITIALIZATION
Run python init_project.py.

STEP 2: INSTALLATION
Create/Activate virtual environment.
Run python install_deps.py (if Terminal is blocked) OR pip install -r requirements.txt.

STEP 3: CONFIGURATION
Paste code into config.json and rules.json.
Place PDFs in input/ or input/_MANUAL_UPLOADS.

STEP 4: INDEXING (TIER 1)
Run python relationship_manager.py.
Generates output/relationship_manifest.json.

STEP 5: EXTRACTION (TIER 1)
Run bootstrap.bat OR python extractor.py.
Generates output/*_v2.json files.

STEP 6: VALIDATION
Run scripts/run_tests.bat.

STEP 7: DEPLOYMENT (TIER 2)
Upload copilot_tier2.py to Copilot Studio.
